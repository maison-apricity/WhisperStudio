# WhisperStudio

WhisperStudio는 OpenAI사의 음성인식 엔진인 Whisper를 사용하여 영상·음성 파일을 텍스트로 전사하고, 바로 `.srt` 자막 파일로 저장할 수 있는 Windows용 데스크톱 애플리케이션입니다.

프로그램은 실행 환경을 점검한 뒤 **가능하면 GPU를 우선 사용하고**, 환경이 지원되지 않는다고 판단될 시 **CPU 모드로 자동 전환**합니다.

모든 파일은 코딩부터 빌드까지 전부 LLM의 도움을 받아 진행하였습니다. 본인은 본 코드에 대해 기초적인 지식 외 아는 것이 전무하므로 궁금증이 생길 경우 AI의 도움을 받으시길 권장합니다.

---

## 무엇을 할 수 있나요?

- 영상·음성 파일 전사
- `.srt` 자막 파일 저장
- 모델 자동 다운로드 및 로컬 캐시 관리
- 장치 상태 점검 후 가능한 실행 조합 자동 선택
- 여러 전사 프리셋 제공
- 작업 로그, 진행 상태, 자원 상태 확인
- 모델이 없을 때 앱 안에서 바로 다운로드
- 여러 파일을 한 번에 선택하여 일괄 전사
- `.srt` 외에 `.vtt`, `.txt` 형식으로도 저장 가능
- 음성 보정 적용 및 실패 시 기본 전처리 또는 원본 입력으로 자동 전환

## 지원 입력 형식

다음과 같은 미디어 파일을 열 수 있습니다.

- `.mkv`
- `.mp4`
- `.mov`
- `.avi`
- `.mp3`
- `.wav`
- `.m4a`
- `.flac`
- `.ogg`
- `.webm`

출력 파일은 입력 파일과 같은 위치에 저장되며, 기본 형식은 **`.srt`**입니다. 필요하면 **`.vtt`**, **`.txt`**도 함께 저장할 수 있습니다.

## 사용법

### 1. 프로그램 실행
`WhisperStudio.exe`를 실행합니다.

### 2. 입력 파일 선택
전사할 영상 또는 음성 파일을 선택합니다.

### 3. 언어 선택
입력 음성의 언어를 지정합니다.  
언어를 명확히 알고 있다면 직접 지정하는 편이 보통 더 안정적입니다.

### 4. 모델 선택
원하는 모델을 선택합니다.

- **Large v3 Turbo**: 기본값, 속도와 품질의 균형
- **Large v3**: 품질 우선
- **Medium / Small / Base / Tiny**: 더 가벼운 실행
- **Distil Large v3**: faster-whisper 친화형 증류 모델

선택한 모델이 로컬에 없으면 앱이 다운로드를 안내합니다.

### 5. 전사 프리셋 선택
콘텐츠 성격에 맞는 프리셋을 선택합니다.

예:
- 균형
- 강의·회의
- 영화·드라마·인터뷰
- 공연·현장
- 속도 우선
- 정확도 우선

### 6. 시작
전사를 시작하면 진행률, 로그, 현재 작업 상태를 확인할 수 있습니다.

## FFmpeg 안내

이 배포본에는 라이센스 이슈로 **FFmpeg를 포함하지 않았습니다.**
**안정적인 미디어 처리와 호환성을 위해 사용자가 직접 FFmpeg를 내려받아 넣으시길 권장합니다.**

[FFmpeg 공식 다운로드 홈페이지](https://www.ffmpeg.org/download.html)

### 권장 폴더 위치

아래 경로에 FFmpeg 실행 파일을 넣어두십시오.

```text
_include/ffmpeg/
```

권장 예시는 다음과 같습니다.

```text
WhisperStudio/
├─ WhisperStudio.exe
├─ _include/
│  └─ ffmpeg/
│     ├─ ffmpeg.exe
│     └─ ffprobe.exe
├─ fonts/
├─ licenses/
└─ settings.json
```


## GPU 사용 방식

WhisperStudio는 내부적으로는 실행 시점에 가능한 경로를 판단합니다.

- GPU를 사용할 수 있으면 GPU 사용
- GPU 초기화에 실패하면 CPU fallback
- 사용자가 원하면 장치 선호를 바꿔서 다시 시도 가능

 **GPU 가속을 사용할 경우 시스템과 실행 환경에 맞는 GPU 런타임 구성이 필요할 수 있습니다.** 


### GPU가 안 잡히는 경우
다음과 같은 원인이 있을 수 있습니다.

- NVIDIA 드라이버가 너무 오래된 경우
- **현재 시스템의 GPU 아키텍처와 번들된 런타임 조합**이 맞지 않는 경우
- 앱이 장치 점검 후 CPU 경로를 더 안정적인 조합으로 판단한 경우

이 경우에도 프로그램을 사용하는 데에는 문제가 없지만, CPU을 사용하여 전사하므로 속도가 저하될 수 있습니다.


### GPU 런타임 호환성 안내

이 배포본은 **PyTorch CUDA 12 계열 + faster-whisper/CTranslate2 기반 GPU 실행**을 전제로 합니다.  
최신 드라이버를 사용하더라도 GPU 아키텍처가 너무 오래된 경우에는 GPU 가속에 실패하고 CPU로 전환될 수 있습니다.  
NVIDIA의 공식 문서를 기반으로 **Turing 계열 이후의 NVIDIA GPU**에서 가장 안정적으로 동작하며, **Pascal·Volta 등 구형 계열**은 조합에 따라 GPU 가속이 되지 않을 수 있습니다.

## 파일 및 폴더 구조

실행 후 생성되거나 함께 둘 수 있는 주요 폴더는 다음과 같습니다.

```text
WhisperStudio/
├─ WhisperStudio.exe
├─ _include/
│  └─ ffmpeg/
├─ hf_home/
├─ temp/
└─ settings.json
```

### 설명
- `WhisperStudio.exe` : 프로그램 파일
- `_include/` : 프로그램 실행에 필요한 모든 파일이 포함된 폴더
- `hf_home/` : 모델 캐시 및 Hugging Face 관련 데이터
- `temp/` : 전사를 위한 오디오 추출 중 생성되는 임시 파일이며, 프로그램 종료 시 가능한 범위에서 정리됩니다.
- `settings.json` : 사용자 설정 저장 파일

## FAQ

### Q. GPU가 있는데도 CPU로 돌아갑니다.
A. 장치 점검 결과, 현재 시스템에서 CPU 조합이 더 안정적인 것으로 판단되었을 수 있습니다. 드라이버 상태와 장치 설정을 확인한 뒤 다시 시도해 보십시오.

### Q. FFmpeg 없이 써도 되나요?
A. 시스템에서 FFmpeg가 확인되지 않을 경우 PyAV를 사용합니다. 다만, 안정적인 작업을 위해 **직접 FFmpeg를 공식 홈페이지에서 다운 받으시길 권장합니다.** 다운로드하신 `ffmpeg.exe`, `ffprobe.exe`를 `_include/ffmpeg/`에 넣어 두십시오.

### Q. 모델 다운로드가 오래 걸립니다.
A. 모델의 크기는 최대 2Gb 가량으로 클 수 있습니다. 최초 다운로드 이후에는 캐시를 활용합니다. 다운로드된 모델은 `hf_home`에 저장됩니다.

### Q. 자막은 어디에 저장되나요?
A. 기본적으로 입력 파일과 같은 위치에 `.srt` 파일로 저장됩니다.

### Q. CPU만으로도 사용할 수 있나요?
A. 가능합니다. WhisperStudio는 GPU가 불가능한 환경에서도 CPU 전사를 지원합니다.

### Q. 음성 보정이 오래 걸리거나 실패하면 어떻게 되나요?
A. 선택한 음성 보정이 제한 시간을 넘기거나 실패하면 앱이 기본 전처리로 자동 전환합니다. 기본 전처리도 실패하면 원본 입력을 사용하여 가능한 범위에서 작업을 계속 진행합니다.
